How to install the application?

  • 1. Download the application (56 Mo) with the button "".
  • 2. Put the file downloaded (VRLanguageLearning.apk) on your smartphone.
  • 3. With your smartphone, select the file and click on "install".
  • 4. Don't forget to activate your microphone and to increase the "multimedia" (or "media") volume, on your phone's parameters.

IMPORTANT: The application works only on Android smartphones which can run VR apps!
